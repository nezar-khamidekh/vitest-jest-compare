# VitestJestCompare

## Описание проекта

Этот проект представляет собой сравнительный анализ производительности двух популярных тестовых фреймворков для Angular приложений:

- **Vitest** - современный и быстрый тестовый фреймворк
- **Jest** - традиционный и широко используемый тестовый фреймворк

## Цели проекта

1. **Сравнение производительности** - измерение времени выполнения тестов в Vitest vs Jest
2. **Анализ совместимости** - проверка корректности работы Vitest с Angular 20
3. **Создание тестового набора** - разработка comprehensive набора unit-тестов для различных Angular компонентов

## Структура тестов

Проект содержит набор unit-тестов для следующих Angular элементов:

### Компоненты (Components)

- Базовые компоненты приложения
- Компоненты с различными жизненными циклами
- Компоненты с зависимостями

### Сервисы (Services)

- HTTP сервисы
- Сервисы состояния
- Сервисы с бизнес-логикой

### Пайпы (Pipes)

- Трансформационные пайпы
- Пайпы форматирования
- Кастомные пайпы

### Директивы (Directives)

- Структурные директивы
- Атрибутные директивы
- Кастомные директивы

## Техническая реализация

### Конфигурация Vitest

- Использует `vitest.config.ts` для настройки
- Поддерживает Angular специфичные настройки
- Интегрирован с Angular CLI через `angular.json`
- **Версия:** Vitest 3.2.4

### Конфигурация Jest

- Использует `jest.config.ts` для настройки
- Применяет `jest-preset-angular` для Angular совместимости
- Настроен для работы с TypeScript и HTML шаблонами
- **Версия:** Jest 30.0.5 с jest-preset-angular 15.0.0

### Структура файлов

```
src/
├── app/
│   ├── app.vitest.spec.ts    # Тесты для Vitest
│   └── app.jest.spec.ts      # Тесты для Jest
├── vitest-test-setup.ts      # Настройки Vitest
└── jest-test-setup.ts        # Настройки Jest
```

## Запуск тестов

### Vitest

```bash
npm run test:vitest
# или
npm test
```

### Jest

```bash
npm run test:jest
```

## Требования

- Node.js 18+
- Angular CLI 20+
- TypeScript 5.8+

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск тестов Vitest
npm run test:vitest

# Запуск тестов Jest
npm run test:jest

# Запуск приложения
npm start
```
